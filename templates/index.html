Abstract

We divide 120 stocks into 3 groups according to the surprise factor which is the ratio of actual eps over estimated eps. Then we calculate the cummulated abnormal return of each group and output the graph with cummulated abnormal return.
EPS data is from Bloomberg and price data is automatically downloaded from Yahoo.

We really appreciate open source resourece provided by Yahoo, Twitter, Flask and etc, without which we cannot build up such beautiful web application.
